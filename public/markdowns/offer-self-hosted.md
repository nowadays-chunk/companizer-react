# Self-Hosted Offer
## Total Control & Data Sovereignty

The **Self-Hosted** plan is the premier choice for organizations with strict data governance requirements, existing infrastructure investments, or a desire for complete control over their ERP environment.

### Commercial Overview
*   **Pricing Model**: Flat Monthly Fee (covers base) + Overage for extra users/entities.
*   **Ideal For**: Regulated industries (FinTech, Healthcare), Enterprises, and detailed-oriented IT teams.
*   **Value Proposition**: Fixed cost for up to 20 users reduces scaling friction.

### Technical Specifications
*   **Deployment Targets**:
    *   On-Premise Servers (Linux/Windows).
    *   Private Cloud (AWS VPC, Azure Private Link, Google Cloud).
    *   Kubernetes / Docker Swarm compatible.
*   **Data Control**: You hold 100% of the encryption keys and data files. No telemetry is sent back to Vault Insight without explicit opt-in.
*   **Updates**: Controlled update release cycle (LTS versions available). You decide when to upgrade.

### Included Features
*   **Base Capacity**: Includes up to **20 Users** and **2 Organizations** in the flat fee.
*   **Customization**: Full access to custom script hooks and API extensions.
*   **Backups**: Configurable custom backup strategies (S3, Local, FTP).
*   **Support**: Priority Ticket Support (12h response time).

### Why Choose Self-Hosted?
*   **Compliance**: Meet GDPR, HIPAA, or internal security mandates with ease.
*   **Performance**: run on your own high-performance hardware.
*   **Integration**: Direct database access for complex internal integrations (SQL/NoSQL).
